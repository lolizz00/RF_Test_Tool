 # RF_TEST_TOOL
 
 
 ###### Синтаксис комитов:
 
  -  #EOFD 		     	     
      - коммиты в конце дня, существенных изменений не несут 
  -  ...-non-tested 	 	   
     - внесены существенные изменения, но не проверены на сборке
  -  ...-released"\*число\*"      
     - отправлено в релиз энного *числа*
  -  ...-fix\*косяк\*		      
     - исправлен \*косяк\*
